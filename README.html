The project is a social network where people can sign up, make friends and share messages and photos with each other.

Parts of the web shop are not finished yet and need to be implemented by you. You are free to use and extend existing parts of the project or implement your own.

Delivery Instructions

At the end create a ZIP file named RM2013_Surname_Prename.zip of your work and copy it to the location given to you by your instructor.

Rating Criteria

Each task gets rated by the implementation of the required functionality. In addition the following criteria apply to all tasks and must match your solution:

All code needs to be nicely formatted, indented and commented where needed. Functions and methods have to be documented within the code (except getter and setter). The generated output is needed to comply to the HTML5 specification. SQL queries have to be protected against SQL injection attacks. No system errors must be display on the website. Database access, input handling and output generation have to be separated. The website must practice good usability principles.
